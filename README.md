# SagX

SagX is a full-stack mission management platform designed for organizations that require structured and semi-automated workflows to replace manual task coordination.

It also includes real-time communication features such as chat, voice/video calls, and screen sharing to improve collaboration between users.

---

## Features

- Secure user authentication system
- Two-Factor Authentication (2FA)
- Refresh token-based session management
- Mission / task management system
- Real-time chat system
- Voice and video calling
- Screen sharing for collaboration
- Role-based workflow structure

---

## Tech Stack

**Frontend:**
- React
- TypeScript
- Tailwind CSS
- CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other:**
- WebSockets (for real-time communication)

---

## Purpose

SagX simulates a real-world enterprise workflow system that combines task management and communication tools in a single platform. It is designed to replicate internal tools used in modern organizations for coordination, collaboration, and mission tracking.
